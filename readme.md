### 文件结构 structure

```plain
--root
  --try_first.py      ;当前版本所有受支持功能的demo
  --src               ;放分模块代码的地方
    --__init__.py     ;常量都放在这里面
    --font.py         ;负责在窗口上绘制字体的模块
    --img.py          ;负责加载图片的模块
    --obj.py          ;负责加载物体的模块
  --Assert            ;放素材的地方
    --font            ;字体
    --pic             ;图片
    --sound           ;音频
  --readme.md         ;公告栏
```



### 受支持的功能 functions provided

- 图片绘制，支持静图/动图绘制、缩放、移动

- 物体绘制，支持移动，使用自驱动思想

- 字体绘制，支持动态改颜色、动态改内容、移动、双层字体（可用于实现阴影效果）

- 键盘按键检测（不太灵光）

- demo中有提供FPS检测、显示
